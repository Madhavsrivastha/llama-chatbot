# 🤖 Llama Chatbot (RAG Based)

## 📌 Overview
This project implements a Retrieval-Augmented Generation (RAG) chatbot using Llama.

## 🚀 Features
- Context-based chatbot
- Vector embeddings
- Question generation

## 🛠 Tech Stack
- Python
- Llama
- Qdrant

## ▶️ How to Run
pip install -r requirements.txt
python generate_valid_questions.py
